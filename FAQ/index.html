Antistasi Ultimate is a mod for Arma 3 that expands on Antistasi Plus with new content, gameplay mechanics, and community-driven features.

An RPT file is a "report" log file. It logs everything happening and can aid developers in helping you or fixing bugs.
You usually find them here (Windows): C:\Users\[name]\AppData\Local\Arma 3
Look for the files with the prefix Arma3_x64_YEAR_MONTH-DAY_HOUR-TIME-TIME.rpt

Generally: C:\Users\[name]\Documents\Arma 3 - Other Profiles\[profilename].
Servers vary on where they are stored, so just search for .vars if possible.
In there, you will need either [profilename].vars.Arma3Profile or AntistasiUltimate.vars.
